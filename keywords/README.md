# keywords

## Purpose
Keyword reference libraries used when drafting recommendations.

## What belongs here
- Keyword lists filed under `master-negatives/`, `exact-match-library/` or `expansion-candidates/`

## What should NOT be stored here
- Keyword strategy rules (use `context/keyword-strategy.md`)
- Raw search term exports (use `evidence/`)
