# skills

## Purpose
Definitions of the repeatable PPC checks the AI can run. Every skill produces DRAFT recommendations only.

## What belongs here
- One markdown file per skill
- The safety wording required of every skill: DRAFT only, never applies Amazon Ads changes, human approval required, STOP if automation is implied

## What should NOT be stored here
- Skill output (use `reports/`, `evidence/` or `decisions/`)
- Executable code or automation scripts
- Business rules (use `context/`)
