# campaigns

## Purpose
Documentation of Amazon PPC campaigns by lifecycle state. Documentation only — this repository never changes campaigns in Amazon Ads.

## What belongs here
- One document per campaign, filed under `active/`, `paused/` or `archived/`

## What should NOT be stored here
- Bid or budget rules (use `context/`)
- Performance exports (use `evidence/`)
- Any instruction intended to be executed automatically
